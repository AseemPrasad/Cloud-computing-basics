# Cloud Storage Setup using Backblaze B2

## Overview
This project demonstrates how to create and configure cloud storage using *Backblaze B2 Cloud Storage*.  
It fulfills the requirements of the Cloud Storage Setup Task:

> CREATE AND CONFIGURE CLOUD STORAGE ON AWS S3 OR GOOGLE CLOUD STORAGE  
> DELIVERABLE: A BUCKET SETUP WITH EXAMPLE FILES UPLOADED AND ACCESS PERMISSIONS CONFIGURED.


## Steps Performed

1. *Created a Backblaze B2 account*
   - Accessed: https://www.backblaze.com/b2/
2. *Created a storage bucket*
   - Name: Aseemcodtechtask1
   - Type: Private
3. *Uploaded a file*
   - Example file: Samplescript1
4. *Configured permissions*
   - Bucket set to private
5. *Verified setup*
   - Screenshots of all steps included
